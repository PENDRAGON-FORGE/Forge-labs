# Flujo de Trabajo Forge Labs

## Regla principal

Cada sesion debe terminar con entregables tangibles.

## Flujo

1. Definir objetivo.
2. Investigar.
3. Documentar.
4. Disenar.
5. Validar.
6. Construir.
7. Probar.
8. Registrar decisiones.

## No permitido

- Programar sin PRD.
- Elegir tecnologia sin justificar.
- Construir funciones sin ROI.
- Crear productos aislados.
